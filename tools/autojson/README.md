# AutoJSON 工具说明文档

该目录下的工具用于通过 `levels.xlsx` 自动生成和更新 `T_异常副本.json` 中的关卡选项。

## 1. 目录结构

- [`更新资源类型.py`](tools/autojson/更新资源类型.py): 核心 Python 脚本，用于读取 Excel 并更新 JSON。
- [`levels.xlsx`](tools/autojson/levels.xlsx): 存储关卡数据的数据源。
- [`T_异常副本.json`](tools/autojson/T_异常副本.json): 脚本生成的本地目标文件（供预览和检查）。

## 2. 工具用途

在游戏中，异常副本的关卡会随版本更新而增加。手动维护 `T_异常副本.json` 中的数百个选项（包括分层菜单、资源分类等）非常繁琐且易出错。
本工具通过 Excel 结构化管理关卡，自动生成复杂的树形 `option` 结构，支持：

- 按 **思绪残影** (角色 > 残影名称) 分类。
- 按 **角色材料** (角色 > 级别) 分类。
- 按 **印象材料** (印象类型 > 级别) 分类。

## 3. 使用方法

1. **环境准备**:
   确保已安装 `openpyxl` 库：

   ```bash
   pip install openpyxl
   ```

2. **编辑数据**:
   在 [`levels.xlsx`](tools/autojson/levels.xlsx) 中添加或修改关卡数据。

3. **运行脚本**:
   在项目根目录下运行（或直接在 `tools/autojson` 目录下运行）：

   ```bash
   python tools/autojson/更新资源类型.py
   ```

4. **应用更改**:
   运行后，脚本会读取 `assets/resource/tasks/T_异常副本.json` 作为模板，在 `tools/autojson/T_异常副本.json` 生成更新后的版本。检查无误后，手动覆盖原文件或根据需要移动。

## 4. `levels.xlsx` 填写要求

Excel 必须包含以下表头字段（第一行）：

| 字段名 | 说明 | 示例 |
| :--- | :--- | :--- |
| `episode_id` | 章节 ID (数字) | `1`, `11` |
| `level_int` | 关卡 ID (数字) | `4`, `12` |
| `session` | 章节部分（part）ID（数字） | `1`, `2` |
| `volume_name` | 对应主线篇章名称 | `剑兰谷篇` |
| `角色` | (可选) 对应角色名 | `夏彦`, `左然`, `莫弈`, `陆景和` |
| `思绪残影` | (可选) 残影名称 | `彼时少年` |
| `角色初级/中级/高级` | (可选) 填写 `1` 表示该关卡产出对应级别的角色材料 | `1` |
| `所属印象` | (可选) 印象类型 | `逻辑`, `共情`, `直觉` |
| `印象I/II/III` | (可选) 填写 `1` 表示产出对应级别的印象材料 | `1` |
| `episode_template` | (可选) 手动指定的模板匹配图片路径 | `Debate/E1P1.png` |
| `max_idx` | (可选) 列表滑动边界标识 (最下) | `1` |
| `min_idx` | (可选) 列表滑动边界标识 (最上) | `1` |

### 注意事项

- **篇章名称与 OCR**: `volume_name` 字段将被直接用于 OCR 识别（即 `expected` 字段）。因此，请确保填写的文字与OCR结果相符。
- **正则表达式支持**: `volume_name` 支持使用正则表达式。
- **自动路径**: 如果未填写 `episode_template`，脚本会自动根据 `episode_id` 和 `session` 生成 `Debate/E{ep_id}P{session}.png` 路径。
- **Session 留空**: 如果 `session` 字段留空且未手动指定模板路径，脚本会将识别方式从“模板匹配”切换为“文本识别”，生成 `expected: "Episode.XX"` (XX 为补零后的章节 ID)。
- **ID 补零**: 脚本会自动将 `episode_id` 和 `level_int` 格式化为两位数（如 `01-08`）。
- **分类优先级 (归类逻辑)**:
  - 如果关卡填写了 **`思绪残影`** 名称，该关卡将 **仅** 出现在 `思绪残影` 分类下。
  - 只有在 **`思绪残影` 为空** 时，关卡才会根据标记出现在 `角色材料` 或 `印象材料` 分类中。
