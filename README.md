基于全新架构的生煎包小助手。图像技术 + 模拟控制，解放双手！
由 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 强力驱动！

## 图形化界面

本项目图形化界面基于 MFAAvalonia，解压后运行 `MFAAvalonia.exe` 即可使用图形化界面。
